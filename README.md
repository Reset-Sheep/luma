# Luma

A minimal, content-first design system for Vue.

## Why Luma?
- Designed for product websites, not dashboards
- Focused on typography, space, and storytelling
- Minimal API, expressive layout
- Web-first, Vue-native

## What Luma is not
- Not a component zoo
- Not a mobile UI kit
- Not a dashboard framework
